Analyseur CV / ATS

Outil local — usage personnel

Outil personnel pour comparer une offre d'emploi et un CV, repérer les mots-clés attendus par les systèmes de suivi de candidature (ATS).

Où en est le projet :

  • ✅ Base de mots-clés ESCO (anglais, tous secteurs) ingérée — 16 903 entrées
  • ✅ Base de mots-clés ROME (France Travail, français) ingérée — 37 506 entrées
  • ✅ Moteur d'extraction de mots-clés (recherche exacte, FR + EN)
  • ✅ Interface web (comparaison annonce/CV, import PDF) — testée de bout en bout

Cette page reste un point d'entrée public (référence pour francetravail.io). L'outil lui-même tourne en local sur Mac (python backend/app.py) — voir le README.md du dépôt pour le lancer.
